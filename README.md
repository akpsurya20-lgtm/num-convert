# ⚙️ NUM_CONVERT

> Real-time multi-base translation matrix — A sleek, instant number system converter.

🔗 **Live Demo:** [Try it on CodePen](https://codepen.io/suryaakp/pen/rajVdYj)

## ✨ Features

- **Real-time Conversion** — Type in any base, all others update instantly
- **4 Number Systems** — Hexadecimal (Base 16), Decimal (Base 10), Octal (Base 8), Binary (Base 2)
- **Input Validation** — Prevents invalid characters per base automatically
- **One-Click Copy** — Copy any converted value to clipboard with visual feedback
- **Dark & Light Themes** — Toggle between themes with preference saved to localStorage
- **Responsive Design** — Works flawlessly on desktop, tablet, and mobile
- **Zero Dependencies** — Pure HTML, CSS, and vanilla JavaScript

## 🚀 Quick Start

Open `index.html` in any modern browser. No server required!

## 📁 Project Structure

| File | Purpose |
|------|---------|
| `index.html` | Semantic markup structure |
| `style.css` | Styling, themes, animations, responsive design |
| `script.js` | Real-time conversion logic, validation, clipboard |
